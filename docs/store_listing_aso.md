## Notes / limits

- Free ASO tools can't measure search volume or exact rank (needs AppTweak / Sensor
  Tower). Above is best-practice + product knowledge.
- Google Play title bans: emojis, ALL CAPS, "best"/"#1"/"free", CTAs. Copy above is clean.
- Short description and full description ARE indexed; long description matters for
  search on Google (unlike Apple).
